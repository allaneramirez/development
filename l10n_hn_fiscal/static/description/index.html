Facturación SAR - Libros para Honduras

Extensión contable para Odoo 16 que automatiza los requisitos del Servicio de Administración de Rentas (SAR) de Honduras.

Libros PDF y Excel PT Compras/Ventas CAI y documentos fiscales Campos SAG y OCE

Configuración

  • Define CAI, secuencias y tipos de documento.
  • Configura libros de ventas y compras con diarios/impuestos filtrados.
  • Establece números SAG en el partner para que se propaguen a la factura.
  • Acceso controlado por el grupo Reportes Fiscales HN.

Reportes disponibles

  • Libro de Ventas / Compras (PDF y Excel).
  • PT Libro Compras con conciliación contra asientos contables.
  • PT Libro Ventas con detalle de productos y condición de pago.
  • Columnas dinámicas derivadas del número de documento (ref).

Campos añadidos

ModeloCampoUso
res.partnernumber_sag_hnNúmero de registro SAG, se copia a las facturas.
account.movenumber_sag_hn / number_oce_hnRequisitos del SAR (SAG y Órdenes de compra exentas).
account.moveCampos auxiliares (ISV por tasa, establecimiento, etc.)Alimentan los libros PDF y PT.

Flujo de trabajo

  1. Completar datos de CAI y tipos de documento en Configuración.
  2. Registrar los números SAG/OCE en clientes y proveedores.
  3. Crear configuraciones de libro (ventas y compras).
  4. Ejecutar los wizards desde Reportes Fiscales de Honduras y elegir PDF, Excel o PT Excel.
